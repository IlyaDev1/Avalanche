О проекте

Система предназначена для анализа метеорологических данных и прогнозирования лавинной опасности в горных регионах. Она использует нейросеть Колмогорова-Арнольда (KAN) для точных прогнозов, что помогает спасательным службам принимать решения.

Цели проекта

  • Моделирование метеодатчиков
  • Прогнозирование лавинной опасности
  • Управление данными в PostgreSQL
  • Мониторинг качества через Grafana

Ключевые особенности

  • Сбор и обработка метеоданных (температура, ветер, осадки)
  • REST API на FastAPI
  • Контейнеризация с Docker
  • Мониторинг с Grafana и Prometheus

Архитектура

Проект реализован как микросервисная архитектура с использованием MQTT, FastAPI и PostgreSQL.
